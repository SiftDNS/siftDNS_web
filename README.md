## üìñ About SiftDNS
SiftDNS was a privacy-first, high-performance DNS resolver designed to block ads, trackers, and malware at the network level. 

Architected and executed by a solo founder, the service performed with a "Zero Logs" policy and enterprise-grade uptime over its 8-month operational lifespan in production.

### Key Features
* **Privacy Focus:** Zero-logging architecture to ensure user anonymity.
* **Performance:** Optimized resolution speeds via HA cloud infrastructure.
* **Security:** Network-level blocking of ads, malicious domains and tracking beacons.
